## ðŸ§  Game Overview
**Nonograms** is a logic puzzle where the player reveals a hidden picture by filling in cells on a grid according to numeric clues.

Game rules:
- Numbers next to rows and columns indicate groups of filled cells.
- Groups are separated by at least one empty cell.
- The player must logically deduce which cells should be filled or left empty.
- When all clues are satisfied, the picture is revealed.
